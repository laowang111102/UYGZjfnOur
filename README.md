# 校园跑腿PHP

## 前言

校园跑腿是一款针对在校大学生的便捷生活服务应用。该项目基于PHP开发，结合了Spring、Springmvc、MyBatis等主流框架，同时接入微信小程序，为用户提供了一个方便快捷的跑腿服务解决方案。

## 内容介绍

校园跑腿主要包括以下几个功能模块：用户模块、任务模块、跑腿员模块、消息模块和支付模块。用户可以通过发布任务，寻找合适的跑腿员来完成诸如取快递、买饭等琐事，提高校园生活效率。跑腿员也可以通过完成任务赚取一定的报酬，实现双赢。

## 技术介绍

- 语言：Java
- 使用框架：Spring、Springmvc，MyBatis，微信小程序
- 前端技术：JS、Vue、CSS3，Uniapp
- 开发工具：IDEA/Eclipse，Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven: apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
